Contributing

Want to add your own embeds? Fork this repository, add your HTML files to the embeds/ folder, and submit a pull request!

Make sure your embeds are:

  • Responsive and mobile-friendly
  • Self-contained (all styles and scripts inline)
  • Accessible (proper ARIA labels, keyboard navigation)
  • Well-commented for easy customization
